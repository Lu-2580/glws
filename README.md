# 果链卫士 - 区块链+NFC果园害虫智能识别与溯源平台

## 项目简介

果链卫士是融合区块链+NFC+AI技术的果园害虫智能识别与全链路可信溯源平台，以"一树一芯、一虫一证、一果一码"为核心，打造从虫情识别、数据上链到防控执行、果品溯源的全闭环生态，为果农、监管、消费者提供一体化解决方案，赋能水果产业高质量发展。

## 核心功能

### 1. 害虫智能识别
- 利用AI技术实时识别果园害虫
- 提供准确的虫情分析和防治建议
- 减少农药使用，提高果品品质

### 2. 区块链溯源
- 基于区块链技术实现果品全生命周期的可信溯源
- 确保信息真实不可篡改
- 为消费者提供透明的产品信息

### 3. NFC技术应用
- 通过NFC标签实现一果一码的唯一标识
- 消费者轻触即可查询果品的全链路信息
- 离线可读技术，方便快捷

### 4. 数据管理分析
- 实时收集和分析果园数据
- 为果农提供科学的种植决策支持
- 数据可视化展示，直观易懂

### 5. 智能防控
- 基于虫情数据自动生成防控方案
- 精准用药，减少农药使用
- 提高防治效率，降低成本

### 6. 多端协同
- 支持果农、监管部门、消费者多端使用
- 实现信息共享和协同管理
- 提升整个产业链的透明度和效率

## 技术优势

### 区块链技术
- 数据不可篡改
- 去中心化存储
- 智能合约自动执行

### NFC技术
- 一果一码唯一标识
- 快速扫码查询
- 离线可读技术

### AI技术
- 实时害虫识别
- 智能预测分析
- 自动化防治建议

## 溯源流程

1. **果园数据采集**：通过智能设备实时采集果园环境数据和虫情信息
2. **数据上链存储**：将采集的数据通过区块链技术进行加密存储
3. **AI智能分析**：利用AI技术分析虫情数据，生成智能防控建议
4. **NFC标签赋码**：为每个果品粘贴NFC标签，实现一果一码的唯一标识
5. **消费者查询**：消费者通过手机轻触NFC标签，即可查询果品的全链路信息

- **构建工具**：Vite 5.4.21
- **CSS预处理器**：原生CSS
- **版本控制**：Git

## 安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/Lu-2580/glws.git
   cd glws
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

4. **构建生产版本**
   ```bash
   npm run build
   ```
